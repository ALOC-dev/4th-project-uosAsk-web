## 🚀 빠른 시작

### 환경설정

- Node.js 18+
- pnpm (권장)

### 설치 및 실행

```bash
# 레포지토리 클론
git clone <repository-url>
cd 4th-project-uoScholar-Client

# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하세요.
